# mccli
NodeJS Minecraft Launcher from the command line!

# Setup
just run these commands:

```
git clone https://github.com/TGGamesYT/mccli.git
cd mccli
npm install
npm run start
```
# Usage
after running it, you can use the arrows to navigate through options, enter to select.
